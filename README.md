# Couponsys
##The coupon service.

---
###Run Service

* Install Dependencies
  + npm install
* Run service
  + node ./bin/www
* Current Views:
  + api/v1/
  + api/v1/coupons
  + api/v1/orders

```
Better with nodemon:
Install: npm install -g nodemon
Run: nodemon .bin/www
```
---
###Test

Run test:

```
gulp test
```

Setup test watch task:

```
gulp test-watch
```
